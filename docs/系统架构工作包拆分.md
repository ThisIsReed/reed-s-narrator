### WP-13 主循环业务闭环强化（阶段A）
**目标范围**
- 解决“模块能力已具备，但真实故事回合尚未闭环”的当前主线短板。
- 将物候更新、知识传播、Agent 裁定、状态结算、持久化审计统一收敛到单一 tick 流程中。
- 将现有 demo 中已能分别展示的亮点，提升为同一条运行链上的真实联动能力，而非分段演示。

**问题背景**
- 当前项目的强项在于模块设计清晰、局部机制完整，但主循环组装层仍偏“骨架化”。
- `phenology/`、`knowledge/`、`agents/`、`orchestrator/`、`persistence/` 已分别具备落地能力，但尚未形成“事件触发 -> 可见知识变化 -> 角色意图 -> 裁定结算 -> 状态落库 -> replay 可验证”的单轮闭环。
- 这会导致项目当前更像“高质量机制原型库”，而不是“可持续推进的故事引擎”。

**优化方案**
1. 在 Orchestrator 中显式分层 tick 流程，固定顺序为：
   `Clock -> Phenology -> EventPool -> Knowledge Update -> Spotlight -> ACTIVE Agent -> PASSIVE Update -> Persistence -> Replay Audit`。
2. 将物候硬约束正式纳入 `NarratorController` 主循环，不再作为独立能力悬置在主流程之外。
3. 将 `KnowledgeAssembler.plan_diffusion()` 对应的传播任务接入真实调度，补齐“事件/事实/认知”在 tick 间推进的执行位置与审计记录。
4. 将 ACTIVE 角色执行从“仅调用注入式 runtime”升级为“统一裁定上下文 + 统一结算回写 + 统一日志落库”的标准回合结构。
5. 为 PASSIVE/DORMANT 角色补齐最小一致性更新规则，避免当前主流程对非 ACTIVE 角色只有分层、缺少充分世界演化语义。
6. 在持久化层补齐闭环所需审计面，保证一个 tick 内的关键中间产物都能被回放与对比，而不只是最终 `WorldState`。
7. 将 demo 升级为“闭环验收场景”，要求一个固定 seed 的样例场景能够完整覆盖物候、隔离、裁定、持久化与回放。

**建议拆分**
- WP-13.1 主循环阶段化重构：
  - 为 `NarratorController` 引入清晰的阶段边界和阶段结果对象，避免跨层逻辑继续堆叠在单一方法中。
- WP-13.2 物候与世界状态联动接线：
  - 将 `apply_phenology()` 接入主循环，并将其 `state_changes/audit_log` 纳入统一 tick 结果。
- WP-13.3 知识传播正式落地：
  - 为传播任务引入待执行队列或等价持久化结构，明确任务创建、成熟、执行、审计的时机。
- WP-13.4 Agent 裁定与结算统一化：
  - 将 CharacterAgent / NarratorJudge / DMAgent / fallback 产物统一纳入标准回合输出模型。
- WP-13.5 闭环验收与 demo 场景：
  - 新增一条固定 seed 的端到端样例，作为开发期与回归期共同使用的验收资产。

**验收标准**
- 固定初始世界与固定 seed 下，连续运行与 checkpoint 恢复后继续运行的最终世界状态完全一致。
- 同一 tick 内，不同角色拿到的知识上下文仍满足授权隔离，且传播延迟行为可被 replay 审计。
- 至少存在一条端到端集成测试，覆盖“物候影响 -> 事件生成 -> 知识变化 -> ACTIVE 角色裁定 -> 状态变更 -> 快照/回放”。
- demo 场景不再只是分段展示，而是能在同一条运行链中观察到上述联动结果。

**关键依赖**
- 直接依赖 WP-05、WP-07、WP-08、WP-09、WP-10。
- 应先于 WP-11、WP-12 执行；否则传播增强与可观测性只会叠加在一个尚未收口的主流程骨架上。

## 5. 当前执行顺序判断
1. 阶段 A 的基础底座 WP-01 ~ WP-06 已完成，说明项目已具备“配置 + 模型 + 确定性内核 + 持久化 + 物候 + LLM 抽象”的核心能力。
2. WP-07 ~ WP-10 已把信息隔离、Agent 链路、主循环骨架与 replay 能力基本补齐，当前新的最高优先级应切换为 WP-13。
3. WP-13 的目标不是新增横向功能，而是把现有亮点整合成一个真实的故事回合闭环，属于阶段 A 的收口工作包。
4. WP-11、WP-12 仍应保持在阶段 B，不宜早于 WP-13；否则会在主流程尚未闭环时继续叠加复杂度。
